## 0. 概述
对于重度依赖 AI 的用户来说，使用按量计费的 API 调用可以获得比官方客户端更好的质量和更灵活的使用体验，例如接入工具调用、长期记忆库和多模型工作流。

本文以 DeepSeek 接入 macOS 端 Cherry Studio 为例，讲述如何将 AI 接入第三方客户端。

> [!NOTE]
> API 与一般的 Coding Plan 或会员体系不互通

## 1. 基本步骤
### 1.1. 创建 API Key
1. 打开[DeepSeek开发平台](https://platform.deepseek.com/)注册或登录账号。
2. 选择左侧菜单中的「充值」
    > [!NOTE]
    > API 需要先付款后使用
3. 支付完成后选择左侧菜单中的「API keys」，点击「创建 API key」
4. 输入 API Key 备注后点击「创建」
5. 拷贝屏幕上生成的 API Key
    > [!NOTE]
    > API Key 仅会完整出现一次，后续不能再次查询，请拷贝完整
    >
    > 此外，API Key 是服务器鉴别用户身份的唯一方式，请不要泄漏，否则可能导致余额被盗刷
---
### 1.2. 配置 AI 客户端
> [!NOTE]
> 在本教程中使用 macOS 端的 Cherry Studio 作为示例，其余客户端配置方式大同小异
1. [点击此处](https://cherryai.com.cn/)进入 Cherry Studio 官方网站，下载并安装 Cherry Studio
2. 打开Cherry Studio，选择左下角的齿轮按钮，进入「设置」，并选择「模型服务」
3. 在设置菜单中提供的模型平台中找到「深度求索」，选择并粘贴刚刚拷贝的 API Key
4. 移除所有预设的模型，点击「获取模型列表」，添加所需的模型
5. 点击 API 密钥输入框右侧的「检测」，选择你要使用的模型，点击「确定」，确保提示「连接成功」
---
### 1.3. 测试对话
1. 选择左侧菜单中头像下方第一个选项，进入「对话」页
2. 点击右侧上方「默认助手」旁的模型选择器，确保当前使用的模型是你要使用的模型
3. 尝试输入「你好」，确保模型正常回复，大功告成
